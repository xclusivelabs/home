What is TinyDeck?

TinyDeck is a compact USB device featuring multiple programmable keys, each with its own small color display. Upload custom images, set shortcuts, or trigger macros — TinyDeck turns any desk into a personalized command center.

The companion configuration tool lets you assign actions, swap images, and update firmware without touching a line of code. Power users can dive deeper with open firmware and full serial access.

Features

  • Per-key color display — show custom images or live data on each key
  • USB plug-and-play — no drivers required on Windows, macOS, or Linux
  • Open firmware — modify and flash your own builds
  • Image Prep Tool — crop and convert images to exact display dimensions
  • Compact form factor — fits any desk setup without taking up space
  • Fully programmable shortcuts and macro sequences

Frequently Asked Questions

Can't find what you're looking for? Contact us.

How do I connect TinyDeck to my computer?
TinyDeck connects via USB-C. Plug it into any available USB port — no drivers are needed on Windows 10/11, macOS, or Linux. It will appear as a standard HID device.
What image dimensions does TinyDeck use?
Each key display is [Placeholder] × [Placeholder] pixels. Use the Image Prep Tool tab on this page to upload, crop, and export your images at the exact correct resolution before loading them onto the device.
What image formats are supported?
TinyDeck accepts PNG and BMP image files. The Image Prep Tool on this page handles resizing and conversion automatically — just upload any common image and download the prepared file.
How do I update the firmware?
Download the latest firmware from the Firmware tab. Hold the BOOT button while connecting TinyDeck via USB to enter bootloader mode, then use the included flashing tool to upload the .bin file. Detailed step-by-step instructions are included in the firmware package.
Is the firmware open source?
Yes. The TinyDeck firmware is open source and can be modified and reflashed freely. Source code and build instructions are provided with the firmware download.
My TinyDeck isn't recognized by my computer. What do I do?
Try a different USB cable (some cables are charge-only) and a different USB port. If the problem persists, try holding the BOOT button during connection to enter bootloader mode and reflash the firmware. If you're still stuck, contact us.

What is Mini Macropad?

Mini Macropad is a compact mechanical keyboard with a small number of fully programmable keys. Each key can be assigned any keyboard shortcut, application launcher, text snippet, or complex multi-step macro sequence.

Whether you're a video editor, developer, streamer, or power user, Mini Macropad puts your most-used actions a single keypress away.

How many keys does Mini Macropad have?
Mini Macropad has [Placeholder] physical keys, each fully programmable. With layer support, the effective number of actions is multiplied by the number of configured layers.
Is it compatible with Windows and macOS?
Yes. Mini Macropad presents itself as a standard USB HID keyboard, so it works on Windows, macOS, and Linux without installing any drivers. The companion configuration software runs on Windows and macOS.
Can I program complex multi-step macros?
Yes. Each key can trigger a sequence of keypresses with configurable delays between each step. You can build macros for text snippets, application launchers, window management, and more through the companion software.
What are layers and how do I use them?
Layers let you assign different actions to the same physical keys by switching "profiles." For example, Layer 1 could be shortcuts for your code editor, and Layer 2 for your video editor. You can switch between layers by holding or tapping a designated key.
How do I update the firmware?
Download the latest firmware from the Firmware tab. Hold the BOOT button while connecting Mini Macropad via USB to enter bootloader mode, then use the included flashing tool to upload the .bin file. Your key mappings are preserved after flashing.
